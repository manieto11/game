# 2D Platformer Game (WIP)

This repository contains an in-progress 2D platformer game built in C++ with raylib.

## Objective

The main goal of this project is to create a complete 2D platformer game that will likely be released for free, with Steam integration included.

## Current Status

- Work in progress
- Core gameplay systems and engine components are under active development

## Tech Stack

- box2d
- C++
- raylib
- Steamworks API integration

## Usage

Compile with `make game`.
Use `make help` to get extra help.
